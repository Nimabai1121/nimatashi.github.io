Smart Plant Monitoring System

Goal: Automate plant health monitoring for better irrigation and growth conditions.

Approach: Soil moisture + DHT22 (temperature/humidity) sensors → microcontroller (ESP32) → real-time monitoring → IoT dashboard.

Automatic Counting & Packing system

Goal: Automate counting and packing process with operator-friendly interface and real-time data monitoring.

Approach: PLC for process control logic (counting, conveyor, actuator control). HMI for operator interaction (start/stop, production count, error logs). SCADA for real-time monitoring, data logging, and supervisory control.


IoT-Based ECG Monitoring System

Goal: Provide a low-cost, real-time ECG monitoring solution with remote accessibility.

Approach: ECG sensor module + microcontroller (ESP32). Signal preprocessing (filtering & amplification). Data transmission via Wi-Fi to IoT platform ( Blynk,). Real-time visualization on dashboard/mobile app

Hydroponic Automation for Sustainable Agriculture in Bhutan

As part of my final year project, I am working on Hydroponic Automation for Sustainable Agriculture in Bhutan. The system leverages PLC-based control integrated with HMI and SCADA to automate water flow, nutrient dosing, and environmental monitoring for hydroponic crops. This work will be done implementing my knowledge in industrial automation, process control, and smart agriculture systems.
